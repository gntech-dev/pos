# Frequently Asked Questions

## General Questions

### What is the POS System?

The POS System is a comprehensive Point of Sale solution designed specifically for businesses in the Dominican Republic. It includes sales processing, inventory management, customer management, and full DGII compliance with NCF generation.

### What are the system requirements?

**Minimum:**
- 2GB RAM
- 10GB storage
- Internet connection
- Modern web browser

**Recommended:**
- 4GB RAM
- 20GB SSD storage
- Stable internet connection
- Chrome/Firefox/Safari/Edge

### Is it web-based or desktop?

It's a web-based application that runs in any modern browser. No installation required for end users.

### Can it work offline?

Currently, it requires internet connection. Offline mode is planned for future releases.
